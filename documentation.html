The Coffee Igloo API Documentation

Purpose

The Coffee Igloo API allows users to create, view, update, and delete healthy iced coffee recipes.

Endpoints

GET /recipes

GET /recipes/{id}

POST /recipes

PUT /recipes/{id}

DELETE /recipes/{id}

Request Examples

Create a new recipe:

POST /recipes
{
    "name": "Vanilla Protein Latte",
    "category": "High Protein",
    "calories": 180,
    "protein": 25,
    "preparation_time": 5
}

Response Examples

{
    "id": 1,
    "name": "Vanilla Protein Latte",
    "category": "High Protein",
    "calories": 180,
    "protein": 25,
    "preparation_time": 5
}
