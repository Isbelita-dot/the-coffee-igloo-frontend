The Coffee Igloo API Documentation

Project Overview

The Coffee Igloo is a full-stack web application that allows users to create, view, update, search and delete healthy coffee recipes through a REST API.

The frontend is built with HTML, CSS and JavaScript, while the backend uses FastAPI and SQLite.

Technologies

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • Axios
  • Python
  • FastAPI
  • SQLite

Available Endpoints

  • GET /recipes
  • GET /recipes/{id}
  • POST /recipes
  • PUT /recipes/{id}
  • DELETE /recipes/{id}
